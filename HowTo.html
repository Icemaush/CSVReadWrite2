Loading & Saving CSV Files

Loading CSV files

To load a CSV file, go to File > Load CSV.
An Open File dialog will appear and allow the user to select a CSV file to load.
Data from the CSV will be loaded into the output table and file information will be displayed.



Saving CSV files

To save a CSV file, go to File > Save CSV.
The currently opened CSV file will be saved.


Saving CSV files as...

To save a CSV file to a new file/location, go to File > Save CSV As.
A Save File dialog will appear and allow the user to select a file to overwrite or another location to save the file to.

Modifying CSV Files

Modifying rows

To modify a row, select a row from the output table. The data from the selected row will appear in the text boxes of the modify table.
After making changes to the data in the modify table click the "Modify" button to replace the data in the selected row of the output table.


Adding rows

To add rows, click the "Add Row" button. A new blank row will be appended to the output table.


Deleting rows

To delete rows, select a row from the output table and click the "Delete Row" button. The selected row will be removed.


Clearing all rows

To clear all rows from the output table, go to Edit > Clear Data. All rows in the output table will be deleted.
